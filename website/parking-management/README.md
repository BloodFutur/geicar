## Quick start

- Clone the repo
- Make sure your Node.js and npm versions are up to date
- Move to this directory
- Install dependencies: `npm install` or `yarn`
- Start the server: `npm run dev` or `yarn dev`
- Open browser: `http://localhost:3000`
